# Cryptography Notes

## Topics

- Encoding
- Encryption
- Hashing
- Symmetric encryption
- Asymmetric encryption
- Public/private keys
- TLS/HTTPS

## Key Concepts

Encoding is reversible and does not protect data.

Encryption protects data using a key.

Hashing is one-way and is used to verify integrity.

## Common Examples

- Base64 is encoding
- AES is symmetric encryption
- RSA is asymmetric encryption
- SHA-256 is a hashing algorithm
- TLS helps secure web communication
